### 넥스트 스텝: GUI로 업그레이드

지금 스킬은 **터미널에서 명령어**로 실행한다.
GUI를 만들면 **브라우저에서 버튼 클릭**으로 실행할 수 있다.

Claude Code에게 한 줄만 말하면 된다:

```
"내 카드뉴스 스킬을 웹 UI로 만들어줘.
주제를 입력하면 버튼 클릭으로 실행되게.
결과 HTML도 바로 미리보기 되면 좋겠어."
```

---

#### 카드뉴스 스킬 GUI

**지금 방식:**
```
터미널 → /내카드뉴스 마케팅 트렌드
```

**GUI 방식:**
```
브라우저 열기 → 주제 입력 → [생성] 클릭 → HTML 미리보기 → [PNG 저장]
```

만드는 법: Claude Code에게 "내 카드뉴스 스킬 기반으로 간단한 HTML 웹앱 만들어줘" 하면
`app.html` 하나로 완성된다. Python 서버 없이 브라우저에서 바로 열린다.

---

#### 릴스 스킬 GUI

**지금 방식:**
```
터미널 → /내릴스 신제품 출시
```

**GUI 방식:**
```
브라우저 열기 → 주제·스타일 입력 → [생성] 클릭
→ HTML 프리뷰 (Human-in-the-loop) → [확인완료] → 화면녹화 안내
```

특히 릴스는 **Human-in-the-loop 체크포인트**를 UI 버튼으로 만들면
"수정" / "확인완료" 두 버튼으로 더 직관적으로 쓸 수 있다.

---

#### GUI 이후 더 나아가면

| 단계 | 설명 |
|------|------|
| 웹앱 | HTML + JS로 브라우저에서 실행 |
| 로컬 앱 | Electron으로 데스크탑 앱화 |
| 팀 공유 | Vercel 배포 → URL 하나로 팀 전체 사용 |
| 모바일 | PWA로 스마트폰에서도 실행 |

> 이 모든 게 Claude Code에게 "만들어줘" 한 마디로 시작된다.
> 스킬 구조는 이미 안다. 이제 껍데기만 바꾸면 된다.

## EXECUTE

**1. 4주 산출물 목록 만들기**

```bash
cat > MY_OUTPUTS.md << 'EOF'
# 4주 챌린지 완주 산출물

## Week 1
- [ ] Claude Code 설치 완료
- [ ] CLAUDE.md 작성 완료

## Week 2
- [ ] 카드뉴스 완성 (주제: _______)
- [ ] PNG 파일 저장 완료

## Week 3
- [ ] 릴스 완성 (주제: _______)
- [ ] MP4 파일 저장 완료

## Week 4
- [ ] /내카드뉴스 스킬 작동 확인
- [ ] /내릴스 스킬 작동 확인
- [ ] 각 스킬 2회 이상 실행 완료
EOF
```

**2. (선택) 카드뉴스 스킬 GUI 만들어보기**

Claude Code에게 아래 프롬프트 입력:

```
내 카드뉴스 스킬(~/.claude/skills/내카드뉴스/)을 기반으로
간단한 웹 UI를 만들어줘.

요구사항:
- 주제 입력창 1개
- [카드뉴스 생성] 버튼
- 생성된 HTML을 오른쪽에 미리보기
- 외부 라이브러리 없이 HTML 파일 하나로

파일명: cardnews-ui.html
```

**3. 결과 확인**

```bash
open cardnews-ui.html
```

브라우저에서 주제 입력 → 버튼 클릭 → 카드뉴스 생성 확인.

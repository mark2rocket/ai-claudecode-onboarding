| 씬 | 헤드 텍스트 | 서브 텍스트 | 장면 설명 | 영상 소스 | 영상 방향 | 애니메이션 | 자막 |

| 1 | AI가 일하게 해라 | 당신은 디렉터다 | 노트북 앞 고민하는 사람 | GIF | "person thinking laptop" (Pexels) | typewriter | 있음 |

### 소스 유형별 제작 흐름 — 2단계 Human-in-the-loop

style.md의 `영상소스` 값에 따라 제작 경로가 달라진다. **모든 소스 유형에 씬 스크립트 검토가 먼저 들어간다.**

**HTML 애니메이션 단독:**
```
브리프 작성 → 씬별 스크립트 생성 (장면 설명·영상 방향 포함 표 출력)
→ ⛔ STOP #1 "스크립트 확인완료 입력해주세요"
→ HTML 생성 → 브라우저 확인 → ⛔ STOP #2
→ 확인완료 → 화면녹화 (Cmd+Shift+5) → MP4
→ [음성/음악 있으면] FFmpeg으로 오디오 합성
```

**GIF 배경 포함:**
```
브리프 작성 → 씬별 스크립트 생성 (표 출력)
→ ⛔ STOP #1 "스크립트 확인완료 입력해주세요"
→ GIF URL 안내 → HTML 생성 (GIF 오버레이)
→ 브라우저 확인 → ⛔ STOP #2 → 확인완료
→ 화면녹화 → MP4 → [음성/음악] FFmpeg 합성
```

**촬영 영상(MP4) 포함:**
```
브리프 작성 → 씬별 스크립트 생성 (표 출력)
→ ⛔ STOP #1 "스크립트 확인완료 입력해주세요"
→ HTML 생성 (video 태그 + 오버레이 텍스트)
→ 브라우저 확인 → ⛔ STOP #2 → 확인완료
→ 화면녹화 → MP4 → [자막 SRT] FFmpeg 자막 합성
→ [음성/음악] FFmpeg 오디오 합성
```

**복합 소스:**
```
브리프 작성 → 씬별 스크립트 생성 (씬별 소스 유형 포함 표 출력)
→ ⛔ STOP #1 "스크립트 확인완료 입력해주세요"
→ 소스별 섹션 HTML 생성
→ 브라우저 확인 → ⛔ STOP #2 → 확인완료
→ 화면녹화 → 소스 유형별 FFmpeg 편집 → MP4
```

- ✓ 음악 볼륨이 음성을 덮지 않음
- ✓ 음성 파일 길이가 영상 길이와 맞음

### FFmpeg 레퍼런스 (음성·자막 합성)

```bash
# 배경음악 합성 (영상 + MP3)
ffmpeg -i video.mp4 -i music.mp3 \
  -filter_complex "[1:a]volume=0.3[a]" \
  -map 0:v -map "[a]" -shortest output.mp4

# SRT 자막 합성
ffmpeg -i video.mp4 -vf "subtitles=subtitle.srt" output_sub.mp4

# 음성 + 자막 동시 합성
ffmpeg -i video.mp4 -i voice.mp3 \
  -vf "subtitles=subtitle.srt" \
  -map 0:v -map 1:a -shortest output_final.mp4
```

- <div class="gif-bg"><img src="[GIF URL]"></div>

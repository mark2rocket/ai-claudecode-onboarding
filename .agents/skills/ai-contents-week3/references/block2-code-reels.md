### 핵심 개념: HTML 애니메이션 = 소스 없는 영상

```
일반적인 영상 제작:
  카메라 촬영 → 편집 앱 → 영상 파일

코딩으로 만드는 영상:
  Claude → HTML/CSS 애니메이션 → 브라우저 → 화면 녹화 → 영상 파일

장점:
  - 카메라, 편집 앱 불필요
  - 텍스트·색상·타이밍 수정이 코드 한 줄
  - 브랜드 색상/폰트 정확히 적용 가능
  - 완벽하게 재현 가능 (매번 같은 결과)
```

---

### 세로형 영상의 기본 스펙

```
릴스 표준 스펙:
┌──────────────────────────────────────────┐
│  크기:   1080 × 1920px (9:16 비율)       │
│  길이:   15초 ~ 60초 (짧을수록 좋음)     │
│  텍스트: 크고 굵게 (모바일 화면 기준)    │
│  여백:   위아래 15% 안전 영역 확보       │
│  속도:   장면 전환 0.3~0.5초             │
└──────────────────────────────────────────┘
```

---

### CSS 애니메이션 3가지 패턴

```
1. fadeIn (텍스트 등장)
   opacity: 0 → 1
   → 핵심 메시지를 부드럽게 보여줄 때

2. slideUp (아래에서 올라오기)
   transform: translateY(50px) → translateY(0)
   → 리스트 항목을 순서대로 보여줄 때

3. highlight (배경 강조)
   background: transparent → 포인트 컬러
   → 핵심 키워드를 강조할 때
```

---

### 화면 녹화 방법 (Mac 기준)

```
방법 1: QuickTime Player (무료, 기본 앱)
  1. QuickTime 열기
  2. 파일 > 새 화면 녹화
  3. 브라우저에서 HTML 전체화면 실행
  4. 녹화 시작 → 애니메이션 재생 → 중지
  5. .mov 파일로 저장

방법 2: 단축키
  Command + Shift + 5 → 선택 영역 녹화

결과 파일: .mov → 인스타그램 업로드 가능
           FFmpeg으로 .mp4 변환도 가능
```

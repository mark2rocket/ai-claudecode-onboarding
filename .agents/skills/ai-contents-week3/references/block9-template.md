### Week 3 전체 복습

```
Block 0: 복습 + 기획의 중요성
  → 영상은 결정할 요소가 많다 → video_brief.md가 핵심

Block 1: 트랙 선택
  → Track 1(릴스) vs Track 2(Remotion) — 목적이 트랙을 결정한다

Track 1 — 릴스 (Block 2~5):
  Block 2: HTML 애니메이션 → 화면 녹화 (소스 제로)
  Block 3: GIF 배경 + 텍스트 오버레이 (움직이는 배경)
  Block 4: 영상 소스 + FFmpeg 합성 (실제 촬영 활용)
  Block 5: SRT 자막 + TTS 음성 → 완성 릴스

Track 2 — Remotion (Block 6~8):
  Block 6: 홍보 영상 (Hook → 문제 → 해결 → CTA)
  Block 7: 온보딩 영상 (단계별 제품 안내)
  Block 8: 강의 영상 (슬라이드 + 코드 애니메이션)
```

### 영상 브리프 템플릿 3종

```
템플릿의 조건:
  1. 완성형   → 채워진 예시가 있어야 쓰기 쉽다
  2. 목적별   → 릴스 / 홍보 / 강의는 구조가 다르다
  3. 교체 가능 → 내용만 바꿔도 새 영상이 나온다

템플릿 3종:
┌─────────────────────────────────────────────┐
│ template-reels.md        릴스 브리프         │
│ template-promo.md        홍보 영상 브리프    │
│ template-lecture.md      강의 영상 브리프    │
└─────────────────────────────────────────────┘

사용법:
  1. 만들고 싶은 영상에 맞는 템플릿 선택
  2. [ ] 로 표시된 항목만 채우기
  3. Claude에게: "[파일]을 읽고 영상 만들어줘"
  4. 완성
```

---

### 릴스 브리프 vs 홍보/강의 브리프 차이

```
릴스 (Track 1):
  - 형식: 세로 9:16, 15~90초
  - 도구: HTML + FFmpeg
  - 핵심 항목: 장면 구성(초 단위), GIF/영상소스, 자막 여부

홍보 영상 (Track 2 - Remotion):
  - 형식: 가로 16:9, 30~60초
  - 도구: Remotion (React)
  - 핵심 항목: Hook 문장, 문제-해결-CTA 구조

강의 영상 (Track 2 - Remotion):
  - 형식: 가로 16:9, 5~20분
  - 도구: Remotion (React)
  - 핵심 항목: 학습 목표, 슬라이드 구성, 코드 예시
```

### 카드뉴스 vs 영상 — 기획 요소 비교

```
카드뉴스 기획 요소 (5가지):
  주제 / 대상 / 장수 / 구성 / 디자인

영상 기획 요소 (훨씬 많다):
  주제    → 무엇을 전달할 것인가?
  형식    → 릴스? 홍보 영상? 강의? 온보딩?
  길이    → 15초? 60초? 5분?
  소스    → 코딩만? GIF? 직접 촬영?
  자막    → 있음/없음, 스타일, 위치
  음성    → TTS? 직접 녹음? 배경음악?
  트랙    → 세로형(9:16)? 가로형(16:9)?
  툴      → HTML? FFmpeg? Remotion?
```

**결론**: 결정할 게 많을수록 기획(브리프)이 더 중요하다.
`video_brief.md` 하나로 이 모든 결정을 미리 해두면 Claude가 추측할 필요가 없어진다.

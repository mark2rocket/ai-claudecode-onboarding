### Remotion이란?

```
Remotion = React 기반 영상 제작 프레임워크

핵심 개념:
  코드 한 줄 = 영상 한 장면
  React 컴포넌트 = 영상의 장면
  useCurrentFrame() = 현재 프레임 번호
  interpolate() = 값을 부드럽게 변환

기존 툴과의 차이:
  CapCut/Canva:   클릭 → 드래그 → 사람 손이 필요
  Remotion:       브리프 파일 → Claude가 코드 작성 → 렌더링

Claude가 처음부터 끝까지 혼자 실행 가능하다.
Week 4에서 한 줄 명령어로 이 전체 과정을 자동화한다.
```

### Remotion으로 만들 수 있는 영상 3가지

```
┌──────────────────────────────────────────────────────┐
│  유형         목적                 길이              │
├──────────────┼─────────────────────┼─────────────────┤
│  홍보 영상   │ 제품/서비스 소개    │ 30~60초         │
│              │ Hook → 문제 → 해결 → CTA             │
├──────────────┼─────────────────────┼─────────────────┤
│  온보딩 영상 │ 신규 사용자 안내    │ 2~3분           │
│              │ 단계별 기능 소개                      │
├──────────────┼─────────────────────┼─────────────────┤
│  강의 영상   │ 지식/기술 전달      │ 5~20분          │
│              │ 슬라이드 + 코드 애니메이션            │
└──────────────┴─────────────────────┴─────────────────┘

공통 도구: Remotion + React + Claude
공통 스펙: 1920×1080 (16:9)
공통 흐름: 브리프 작성 → Claude에게 → 컴포넌트 생성 → 렌더링
```

### Remotion 설치 (한 번만 하면 됨)

```
1. Node.js 버전 확인 (18 이상 필요)
   node -v

2. ffmpeg 확인
   ffmpeg -version
   없으면: brew install ffmpeg

3. Remotion CLI 전역 설치
   npm i -g @remotion/cli

4. 새 프로젝트 생성
   npx create-video@latest
   → 프로젝트 이름 입력 → Hello World 템플릿 선택

5. 개발 서버 실행
   cd [프로젝트명]
   npm run dev
   → 브라우저에서 영상 미리보기 확인
```

---

### 브리프 → 영상 흐름

```
내가 할 것:
  1. 영상 유형 선택 (홍보 / 온보딩 / 강의)
  2. 자연어로 브리프 설명

Claude가 할 것:
  1. 브리프 파일(.md) 생성
  2. Remotion 컴포넌트 코드 작성
  3. 미리보기 확인 안내
  4. MP4 렌더링
```

## EXECUTE

**Step 0 — Remotion 설치**

```
Claude에게 입력:
"Remotion 사용에 필요한 패키지를 설치해줘.
 Node.js 버전 확인 후 ffmpeg 여부 확인하고
 npx create-video@latest로 프로젝트를 만들어줘.
 프로젝트 이름: my-video"
```

---

**Step 1 — 만들 영상 유형 선택 후 자연어로 요청**

아래 3가지 중 하나를 선택해 Claude에게 입력한다.

**홍보 영상 (30~60초):**
```
[내 제품/서비스 이름] 홍보 영상 60초짜리 만들어줘.
타깃은 [대상], 핵심 가치는 '[가치 한 줄]'.
훅 → 문제([문제 한 줄]) → 해결(가치 3가지) → CTA.
다크 배경, 포인트 컬러는 [색상].
promo_brief.md 만들고 Remotion 컴포넌트 바로 생성해줘.
```

**온보딩 영상 (2~3분):**
```
[내 제품/서비스] 신규 사용자 온보딩 영상 3분짜리 만들어줘.
핵심 기능 3가지: [기능1], [기능2], [기능3].
각 단계에 설명 텍스트 + 화살표 포인터 애니메이션.
onboarding_brief.md 만들고 Remotion 컴포넌트 바로 생성해줘.
```

**강의 영상 (5~10분):**
```
[강의 주제] 강의 영상 5분짜리 만들어줘.
타깃: [수강생 설명].
핵심 3가지: [주제1], [주제2], [주제3].
다크 배경(#1a1a2e), 보라색 포인트, 코드 타이핑 애니메이션.
lecture_brief.md 만들고 Remotion 컴포넌트 바로 생성해줘.
```

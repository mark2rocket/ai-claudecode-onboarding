### 📦 Memory는 두 가지다

```
┌─────────────────────────────────────────────────┐
│  Claude의 기억 시스템                           │
│                                                 │
│  1. CLAUDE.md ─── 내가 쓰는 지시문             │
│     "항상 존댓말로"                              │
│     "내 채널은 30대 직장인 재테크"               │
│     "표로 정리해줘"                              │
│       → 매 대화 시작 시 Claude가 자동으로 읽음  │
│                                                 │
│  2. Auto Memory ── Claude가 스스로 적는 메모    │
│     "새암은 pnpm을 선호함"                       │
│     "이 채널은 7분 이내 영상"                    │
│     "지난번 이 오류는 이렇게 해결됨"             │
│       → 일하면서 발견한 것들을 자동 기록        │
└─────────────────────────────────────────────────┘
```

**한 줄 요약:**
- CLAUDE.md = **내가** Claude에게 주는 규칙서
- Auto Memory = **Claude가** 일하면서 스스로 쓰는 업무 수첩

> 둘 다 매 대화 시작 시 자동으로 읽힌다.
> 따로 설정하거나 외울 필요 없다. 그냥 쓰면 된다.

### 📝 CLAUDE.md 잘 쓰는 법 — 4단계

> 유저 CLAUDE.md는 **150자 이내**가 좋다. 매 대화마다 읽히기 때문에 짧고 핵심만 담는 것이 중요하다.

**1단계: 나는 누구인지**

AI가 가장 먼저 알아야 할 것은 대화 상대의 정체성이다.
개발자인지 비개발자인지에 따라 AI의 설명 수준이 달라진다.

```
- User: 유튜브 크리에이터 (재테크/라이프스타일 채널, 구독자 3만)
- Dev Level: 비개발자. 터미널은 익숙해지는 중, 코드는 쓰지 않음
- Expertise: 영상 기획, 섬네일 디자인, SNS 운영
```

이 세 줄만으로 AI는 기술 용어를 실무 언어로 번역해서 말해주기 시작한다.

**2단계: 어떻게 일하는지**

여기서 핵심은 **역할 분담**이다.

```
- 기술적 결정은 AI가 한다. 콘텐츠 방향 결정은 내가 한다.
- 코드를 직접 짜라고 시키지 말 것. AI가 직접 실행하고 결과만 보고할 것.
- 비즈니스 임팩트 먼저, 기술 디테일은 그다음.
```

**3단계: 자주 쓰는 도구와 맥락**

```
- 주요 도구: Notion, Canva, CapCut
- 채널: 유튜브 + 인스타그램 릴스 동시 운영
- 글쓰기: 한국어 반말 (친근한 톤)
- 고객 정보: 반드시 익명화
```

**4단계: 절대 하지 말아야 할 것**

AI가 잘못된 방향으로 가는 것을 방지한다.

```
- 여러 옵션을 나열하며 "어떤 걸 선택하시겠어요?"라고 묻지 말 것
  → 추천안을 제시하고, 반대 없으면 진행할 것
- 터미널 명령어를 "복사해서 붙여넣으세요"라고 하지 말 것
  → 직접 실행할 것
```

## EXECUTE

세 가지를 순서대로 해보자:

**1단계 — CLAUDE.md 만들기**

```
/init
```

> Claude가 지금 폴더를 분석해서 CLAUDE.md를 자동 생성해준다.
> 생성된 파일을 열어서 내용을 살짝 구경해보자.

**2단계 — 나를 기억시키기**

Claude에게 그냥 말해보자:

```
나는 [이름]이고, [내 직업/역할]이야.
내가 자주 쓰는 말투는 [예: 친근하고 간결하게]야.
이걸 기억해둬.
```

> Claude가 Auto Memory에 기록하는 것을 확인해보자.

**3단계 — 기억 확인하기**

```
/memory
```

> CLAUDE.md와 Auto Memory 파일을 확인할 수 있다.
> 방금 말한 내용이 저장돼 있는지 직접 눈으로 확인해보자.

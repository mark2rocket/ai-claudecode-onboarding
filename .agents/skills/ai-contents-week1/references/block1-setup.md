### 에디터 — 선택사항

파일을 눈으로 보면서 작업하고 싶다면 에디터를 설치할 수 있다:

| 에디터 | 특징 | 추천 대상 |
|--------|------|-----------|
| [Antigravity](https://antigravity.google/) | 무료, AI 내장, 가장 간단 | 처음 시작하는 분 |
| [Cursor](https://www.cursor.com/) | AI 내장, CC 시너지 | 코드 편집도 할 분 |
| [VSCode](https://code.visualstudio.com/) | 범용적, 확장 풍부 | 이미 쓰고 계신 분 |

> 이 챌린지에서는 에디터 없이 터미널에서 `claude`만 쳐도 충분합니다.

### Mac 꿀팁: 숨겨진 폴더 보기

`.claude`, `.git` 처럼 `.`으로 시작하는 폴더는 Mac에서 기본적으로 안 보인다.
Claude에게 시키면 된다:

```
Finder에서 .claude, .git 처럼 숨겨진 폴더도 볼 수 있게 설정해줘
```

그다음 Finder로 직접 확인:
```
open ./
```

> `.claude/skills/` 폴더 안에 지금 배우고 있는 이 교안이 들어있다!

## EXECUTE

순서대로 실행해보자:

1. **첫 실행**: 터미널에서 `claude` 입력
2. **첫 대화**: 아래를 그대로 입력

```
안녕, 나는 [이름]이고 [직업]이야. 나한테 인사해줘
```

3. **5분 자유 대화**: 아무 말이나 해보자. 뭘 물어봐도 된다.

**Claude의 작업 공간(컨텍스트)은 유한하다.**
큰 작업을 메인 대화에서 처리하면 공간이 밀린다.
Subagent는 독립된 공간에서 그 일만 전담하고, 결과만 돌려준다.

### Subagent란?

```
┌─ 메인 Claude ───────────────────────────┐
│  대화: [기획][리서치][제목]... (공간 차고 있음)│
│                                         │
│  "경쟁사 인스타 분석해줘" ──┐            │
│                             ▼           │
│  ┌─ Subagent ──────────────────┐        │
│  │ 시작: [빈 공간]              │        │
│  │ 할 일: 게시물 100개 분석     │        │
│  │ 결과: 인기 패턴 3가지 ────────┼──▶ 전달│
│  └─────────────────────────────┘        │
│  메인 대화: 영향 없음                    │
└─────────────────────────────────────────┘
```

**비유:** 막내 직원에게 "이것만 조사해줘"라고 시키고, 결과만 받는 것.
메인 대화는 건드리지 않고, 작업만 깔끔하게 처리된다.

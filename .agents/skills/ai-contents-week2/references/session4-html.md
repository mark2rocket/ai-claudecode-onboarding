### 왜 HTML인가 (복습)

```
HTML 하나로 다 된다:
  → 브라우저에서 바로 확인 (지금 이 단계)
  → PNG로 저장 → 인스타 업로드 (Session 5)
  → PDF로 변환 → 자료 배포
  → GIF/MP4 → 릴스·유튜브
```

스펙을 파일로 저장해둔 이유도 여기 있다.
다음번에 주제만 바꿔서 "content_brief.md 참고해서 새로 만들어줘" 하면 끝이다.

### 확인 포인트

브라우저에서 열렸을 때 체크:

```
□ 색상·폰트가 스펙대로 적용됐나?
□ 넘버링이 본문 카드에 있나?
□ 헤드라인이 15자 이내로 잘렸나?
□ CTA 카드가 버튼 스타일인가?
□ 전체적으로 "내 브랜드처럼" 보이나?
```

마음에 안 드는 부분이 있으면 그냥 말하면 된다:
```
"2번 카드 배경을 좀 더 어둡게 해줘"
"헤드라인 폰트 크기 더 키워줘"
"넘버링 위치를 오른쪽 상단으로 옮겨줘"
```

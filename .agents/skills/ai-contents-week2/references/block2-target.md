## EXPLAIN

### 🎬 4컷 만화로 시작

```
╔════════════════════╗  ╔════════════════════╗
║  1. 같은 주제      ║  ║  2. 다른 독자      ║
║                    ║  ║                    ║
║  "AI 트렌드 카드뉴스"║  ║  독자 A: 초등학생  ║
║                    ║  ║  → 쉬운 단어,      ║
║  (・・?) 새암      ║  ║     그림 중심      ║
║  "같은 주제인데    ║  ║                    ║
║   누구한테         ║  ║  독자 B: 30대 직장인║
║   쓰는 건지에 따라 ║  ║  → 실무 용어,      ║
║   완전 달라지잖아" ║  ║     데이터 중심    ║
╚════════════════════╝  ╚════════════════════╝

╔════════════════════╗  ╔════════════════════╗
║  3. 장수의 차이    ║  ║  4. 결과 비교      ║
║                    ║  ║                    ║
║  3장 → 핵심만      ║  ║  (≧▽≦) 새암       ║
║  5장 → 적당한 깊이 ║  ║  "대상과 장수를    ║
```

**독자가 없으면 말투도, 깊이도, 용어도 정해지지 않는다.**
**장수가 없으면 1장이 나올 수도, 20장이 나올 수도 있다.**

---

### 예측 훈련

실행 전에 먼저 예측한다. 직접 실행 후 맞는지 확인하는 것이 핵심이다.

**추가되는 것:**
```
Block 1: "AI 트렌드에 대한 카드뉴스 이미지 만들어줘"
Block 2: "30대 직장인을 위한 AI 트렌드 카드뉴스 이미지 5장 만들어줘"
          ↑ 대상 추가                              ↑ 장수 추가
```

**예측해보기:**
```
□ 말투가 달라질 것 같다?
□ 사용하는 용어가 달라질 것 같다?
□ 내용의 깊이가 달라질 것 같다?
```

---

## EXECUTE

**실행 순서:**

1. 실행 전: "Block 1과 뭐가 달라질지" 예측 메모
2. 아래 프롬프트 실행:

```
30대 직장인을 위한 AI 트렌드 카드뉴스 이미지 5장 만들어줘
```

3. Block 1 스크린샷과 나란히 비교
4. 예측이 맞았는지 확인
5. 새 결과 스크린샷 저장

> 비교 방법: 브라우저 탭 두 개 열기 또는 화면 분할

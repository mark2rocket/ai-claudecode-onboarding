### HTML → PNG → SNS 흐름

```
cardnews.html
    ↓ Claude Code (Puppeteer 자동 캡처)
card-01.png  →  인스타그램 1번 슬라이드
card-02.png  →  인스타그램 2번 슬라이드
card-03.png  →  인스타그램 3번 슬라이드
card-04.png  →  인스타그램 4번 슬라이드
card-05.png  →  인스타그램 5번 슬라이드
```

**PNG 스펙 (인스타그램 최적):**
```
크기:    1080×1080px (정사각형)
해상도:  72dpi 이상
파일명:  card-01.png, card-02.png ... 순서대로
```

### OSMU — HTML 하나로 다 된다

PNG 저장까지 완료했다면, 같은 HTML로 다른 포맷도 가능하다:

```
cardnews.html
  → PNG  완료! (인스타그램)
  → PDF  브라우저 인쇄 → 자료·리포트
  → PPT  HTML 슬라이드 → 발표 자료
  → GIF  Week 3에서 → 릴스·유튜브 영상
```

한 번 잘 만들어두면 채널별로 변환해서 계속 쓸 수 있다.

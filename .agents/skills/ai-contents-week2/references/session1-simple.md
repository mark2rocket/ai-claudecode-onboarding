### 지금 실행할 프롬프트

```
AI 트렌드에 대한 카드뉴스 이미지 만들어줘
```

이 한 문장에서 Claude가 스스로 결정하는 것들:

```
주제   → "AI 트렌드"만 줬다. 어떤 트렌드? Claude가 선택.
대상   → 안 줬다. Claude가 결정.
장수   → 안 줬다. Claude가 결정.
구성   → 안 줬다. Claude가 결정.
디자인 → 안 줬다. Claude가 결정.
```

---

## EXECUTE

**실행 순서:**

1. 위 프롬프트를 Claude에게 입력
2. 결과 스크린샷 저장 (이후 블록과 비교용)
3. 아래 체크리스트로 평가:

```
□ 대상이 명시됐나?        → Session 2에서 추가
□ 장수가 내 의도와 맞나?   → Session 2에서 추가
□ 각 카드 구성이 통일됐나?  → Session 4에서 추가
□ 디자인이 내 스타일인가?   → Session 5에서 추가
```

> 체크 못 한 항목이 많을수록 정상이다.
> Session 2~5에서 하나씩 채워가면서 결과가 달라지는 걸 직접 본다.

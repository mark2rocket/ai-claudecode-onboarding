### Session 1~5가 이미 파이프라인이었다

```
Session 1 → Phase 1: 주제 + 대상 + 분량
Session 2 → Phase 2: 스토리 구조 + 후킹
Session 3 → Phase 3: 디자인 스펙 + 레이아웃
Session 4 → Phase 4: HTML 생성
Session 5 → Phase 5: PNG 저장
```

이것을 **한 문서(content_brief.md)에 모아서** 순서대로 실행하는 것이 이 세션의 목표다.

> "자동화는 수동을 이해한 다음에"
> Week 4에서 이 파이프라인을 한 줄 명령어로 만든다.

# 카드뉴스 브리프

제목: "내 직장이 AI로 바뀐다 — 지금 알아야 할 것"

3장: 당장 써볼 수 있는 AI 도구 3가지

5장: 마무리+CTA

## 3. 디자인 스펙 + 레이아웃
크기: 1080×1080px
배경색: #6B4EFF, 텍스트: #FFFFFF
폰트: Noto Sans KR, 헤드라인 15자 이내
레이아웃:
- 1장(표지): 전체 중앙 정렬, 상하 여백 120px
- 2~4장(본문): 좌측 상단 넘버링, 제목 상단 고정, 핵심 요약 하단 고정
- 5장(CTA): 전체 중앙 정렬, 버튼 스타일, 여백 최대화

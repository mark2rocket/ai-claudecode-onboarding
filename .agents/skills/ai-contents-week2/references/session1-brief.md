### 프롬프트 진화

```
1단계 (주제만):
"AI 트렌드에 대한 카드뉴스 만들어줘"
→ 대상, 장수, 주제 폭 전부 Claude 결정

2단계 (주제 + 대상 + 분량):
"30대 직장인을 위한 AI 트렌드 카드뉴스 5장 만들어줘"
→ Claude가 결정할 것이 확 줄어든다
```

**대상이 결과를 바꾸는 이유:**
```
초등학생   → 쉬운 단어, 그림 중심
30대 직장인 → 실무 용어, 데이터 중심
CEO       → 의사결정 관점, 비즈니스 임팩트
```

## EXECUTE

**1단계 — 날것으로 실행:**

```
AI 트렌드에 대한 카드뉴스 만들어줘
```

결과 스크린샷 저장 후 체크:
```
□ 대상이 내가 원하는 사람인가?
□ 장수가 내 의도와 맞나?
□ 내용의 깊이가 맞나?
```

**2단계 — 대상 + 분량 추가:**

```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장 만들어줘
```

> 1단계 결과와 나란히 비교. 세 가지 체크 항목이 얼마나 달라졌는지 확인.

### Block 1~5가 이미 파이프라인이었다

```
Block 1 → Phase 1: 주제 정의
Block 2 → Phase 2: 대상 + 분량
Block 3 → Phase 3: 스토리 구성
Block 4 → Phase 4: 내부 구조 + 포맷
Block 5 → Phase 5: 디자인 스펙
```

이것을 **한 문서(content_brief.md)에 모아서** 순서대로 실행하는 것이 이 블록의 목표다.

> "자동화는 수동을 이해한 다음에"
> Week 4에서 이 파이프라인을 한 줄 명령어로 만든다.

# 카드뉴스 브리프

## 1. 주제
AI 트렌드

## 2. 대상 + 장수
30대 직장인, 5장

- 1~4장: 제목 + 핵심 내용 1~2줄 + 핵심 요약 한 줄
- 5장: 제목 + 핵심 내용 + 행동 유도 문구

## 5. 디자인 스펙
- 크기: 1080×1080px
- 배경색: #6B4EFF
- 텍스트 색: #FFFFFF
- 폰트: Noto Sans KR
- 헤드라인 글자수: 15자 이내

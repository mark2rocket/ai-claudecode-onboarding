### 디자인 스펙 4요소와 각각의 근거

```
┌───────────────────────────────────────────────────┐
│  요소      값          이유                       │

└───────────────────────────────────────────────────┘
```

---

### 내 스펙 결정하기

브랜드 컬러가 있으면 사용한다. 없으면 아래 기본값을 사용한다.

```
크기:   1080×1080px  (변경 불필요)
배경색: #6B4EFF      (보라 계열 — 또는 내 브랜드 컬러)
텍스트: #FFFFFF      (흰색)
폰트:   Noto Sans KR (또는 Pretendard)
헤드라인 글자수: 15자 이내
```

## EXECUTE

**Step 1 — 내 스펙 결정:**

```
배경색: _______  (브랜드 컬러 또는 #6B4EFF)
폰트:   _______  (Noto Sans KR 또는 Pretendard)
```

**Step 2 — 프롬프트 실행:**

```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장을 HTML로 만들어줘.
1장: 표지, 2~4장: 본론, 5장: CTA.
1~4장은 제목+핵심 내용 1~2줄+핵심 요약 한 줄,
5장은 제목+핵심 내용+행동 유도 문구.
1080×1080px 정사각형, 배경색 #6B4EFF, 흰색 텍스트,
헤드라인 15자 이내, Noto Sans KR 폰트
```

> 내 스펙으로 바꿔서 실행해도 좋다.

**Step 3 — Session 4 HTML 결과와 비교:**
- 디자인 스펙이 적용됐는지 확인
- "스펙이 디자인을 바꾼다"는 것을 눈으로 확인

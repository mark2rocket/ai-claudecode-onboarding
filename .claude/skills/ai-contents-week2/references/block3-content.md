---

### 카드뉴스 스토리 구조 틀

```
┌─────────────────────────────────────────┐
│  카드뉴스 기본 스토리 구조 (5장 기준)   │
│                                         │
│  1장: 문제제기   "왜 이걸 봐야 하나?"  │
│  2장: 본론 1     핵심 내용 첫 번째     │
│  3장: 본론 2     핵심 내용 두 번째     │
│  4장: 해결·인사이트  "그래서 어떻게?"  │
│  5장: 행동(CTA)  "지금 당장 이것만"   │
└─────────────────────────────────────────┘
```

이 슬롯에 내 주제를 배정하면 Claude가 내용을 채운다.

**예시:**
```
1장: 표지 (제목 + 부제)
2장: AI가 바꾸는 직장 생활
3장: 당장 써볼 수 있는 AI 도구 3가지
4장: AI 시대 직장인 생존 전략
5장: 마무리 + CTA
```

## EXECUTE

**Step 1 — 내 스토리 구조 설계:**

아래 템플릿을 채워보자:

```
1장 (문제제기): ___________
2장 (본론 1):   ___________
3장 (본론 2):   ___________
4장 (해결):     ___________
5장 (CTA):      ___________
```

**Step 2 — 프롬프트 실행:**

```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장 만들어줘.
1장: 표지 (제목+부제), 2장: AI가 바꾸는 직장 생활,
3장: 당장 써볼 수 있는 AI 도구 3가지,
4장: AI 시대 직장인 생존 전략, 5장: 마무리+CTA
```

> Step 1에서 설계한 내용으로 직접 바꿔도 좋다.

**Step 3 — Block 2 결과와 비교:**
- 스토리 흐름이 생겼는지 확인
- 각 카드가 "할 말"이 생겼는지 확인

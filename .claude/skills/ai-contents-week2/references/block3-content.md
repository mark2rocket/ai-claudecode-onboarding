### 카드뉴스 스토리 구조 틀

```
┌─────────────────────────────────────────┐
│  카드뉴스 기본 스토리 구조 (5장 기준)   │
│                                         │
│  1장: 문제제기   "왜 이걸 봐야 하나?"  │
│  2장: 본론 1     핵심 내용 첫 번째     │
│  3장: 본론 2     핵심 내용 두 번째     │
│  4장: 해결·인사이트  "그래서 어떻게?"  │
│  5장: 행동(CTA)  "지금 당장 이것만"   │
└─────────────────────────────────────────┘
```

이 슬롯에 내 주제를 배정하면 Claude가 내용을 채운다.

**예시:**
```
1장: 표지 (제목 + 부제)
2장: AI가 바꾸는 직장 생활
3장: 당장 써볼 수 있는 AI 도구 3가지
4장: AI 시대 직장인 생존 전략
5장: 마무리 + CTA
```

### 콘텐츠 인풋 2가지 트랙

스토리 구조가 정해지면, **내용을 어디서 가져오느냐**에 따라 두 가지 트랙으로 나뉜다.

```
┌─────────────────────────────────────────────────────┐
│  Track A: 리서치 트랙                               │
│                                                     │
│  Claude Code가 직접 최신 정보를 리서치해서 채운다   │
│                                                     │
│  → 빠르다. 최신 트렌드·데이터·사례를 자동 수집     │
│  → "AI 트렌드 최신 내용으로 채워줘"                │
│  → 크리에이터가 모르는 분야도 커버 가능            │
├─────────────────────────────────────────────────────┤
│  Track B: 인터뷰 트랙                               │
│                                                     │
│  Claude Code가 나에게 질문해서                      │
│  내 경험·노하우로 내용을 만든다                     │
│                                                     │
│  → 차별화된다. 나만 아는 이야기가 들어간다          │
│  → "내 경험을 인터뷰해서 내용을 뽑아줘"            │
│  → 팔로워가 "이 사람만 알고 있는 것"을 느낀다      │
└─────────────────────────────────────────────────────┘
```

**언제 어떤 트랙을 쓰나:**
```
리서치 트랙  → 트렌드·정보성 콘텐츠 (AI 뉴스, 통계, 사례)
인터뷰 트랙  → 경험·노하우 콘텐츠 (내가 써본 후기, 나만의 방법)
```

## EXECUTE

**트랙 선택 후 프롬프트 실행:**

**Track A (리서치 트랙):**
```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장 만들어줘.
1장: 표지 (제목+부제), 2장: AI가 바꾸는 직장 생활,
3장: 당장 써볼 수 있는 AI 도구 3가지,
4장: AI 시대 직장인 생존 전략, 5장: 마무리+CTA.
최신 트렌드와 실제 사례를 리서치해서 내용을 채워줘.
```

**Track B (인터뷰 트랙):**
```
30대 직장인을 위한 AI 활용 경험 카드뉴스 5장을 만들려고 해.
1장: 표지, 2~4장: 내 실제 경험, 5장: CTA.
내용을 채우기 위해 나에게 질문해줘.
내 답변을 바탕으로 카드뉴스 내용을 만들어줄 거야.
```

> 둘 다 실행해보고 톤 차이를 비교해보면 더 좋다.
> 스토리 구조 상세 설계는 Block 4에서 한다.

**Block 2 결과와 비교:**
- 스토리 흐름이 생겼는지 확인
- 각 카드가 "할 말"이 생겼는지 확인
- 두 트랙의 결과물 톤 차이를 비교

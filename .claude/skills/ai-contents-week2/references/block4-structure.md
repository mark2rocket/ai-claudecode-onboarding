---

### 카드 내부 구성 공식

```
┌──────────────────────────────────────────┐
│  1~4장 공식                              │
│  제목          (15자 이내)               │
│  핵심 내용     (1~2줄)                   │
│  핵심 요약     (한 줄 — 이 카드의 핵심)  │
│                                          │
│  5장 공식 (CTA)                          │
│  제목          (15자 이내)               │
│  핵심 내용     (1~2줄)                   │
│  행동 유도 문구 (지금 바로 ~하세요)      │
└──────────────────────────────────────────┘
```

> CTA는 마지막 카드에만 집중한다.
> 1~4장에 강한 CTA를 넣으면 독자의 주의가 분산된다.

---

### HTML 포맷

HTML로 지정하면 브라우저에서 실제 카드처럼 볼 수 있다.

```
텍스트 출력  →  그냥 글자 나열
HTML 출력   →  카드 레이아웃, 색상, 여백이 적용된 시각적 결과물
```

## EXECUTE

**프롬프트 실행:**

```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장을 HTML로 만들어줘.
1장: 표지, 2장: AI가 바꾸는 직장생활, 3장: AI 도구 3가지,
4장: 생존 전략, 5장: CTA.
1~4장은 제목+핵심 내용 1~2줄+핵심 요약 한 줄로,
5장은 제목+핵심 내용+행동 유도 문구로 구성해줘
```

**HTML 파일 저장 및 열기:**

```
1. Claude가 생성한 HTML 코드 전체 복사
2. 텍스트 편집기(메모장, VS Code 등)에 붙여넣기
3. 파일명 cardnews.html로 저장
4. 파일 더블클릭 → 브라우저에서 열기
```

> Block 3 결과와 비교: 각 카드 내부가 통일됐는지 확인

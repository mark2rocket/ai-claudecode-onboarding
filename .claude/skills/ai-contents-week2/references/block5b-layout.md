### 레이아웃 3요소

```
┌─────────────────────────────────────────────┐
│  레이아웃 3요소                             │
│                                             │
│  1. 계층  헤드라인 > 본문 > 요약의          │
│           크기 비율 (예: 48px / 24px / 18px)│
│                                             │
│  2. 여백  카드 내부 padding, 요소 간 간격   │
│           (예: 상하 120px, 좌우 80px)       │
│                                             │
│  3. 배치  텍스트 정렬, 넘버링 위치,         │
│           요약 고정 위치 등                 │
└─────────────────────────────────────────────┘
```

### 카드 유형별 레이아웃

```
┌──────────────────────────────────────────────────┐
│  유형        레이아웃 특징                       │
├──────────────┬───────────────────────────────────┤
│  1장 (표지)  │ 제목 중앙 정렬, 상하 여백 넉넉히  │
│              │ 부제는 제목 크기의 50%            │
├──────────────┼───────────────────────────────────┤
│  2~4장 (본문)│ 좌측 상단 넘버링                  │
│              │ 제목 상단 고정                    │
│              │ 본문 중앙 배치                    │
│              │ 핵심 요약 하단 고정               │
├──────────────┼───────────────────────────────────┤
│  5장 (CTA)   │ 전체 중앙 정렬                    │
│              │ CTA 문구 버튼 스타일              │
│              │ 여백 최대화 (임팩트 강조)         │
└──────────────┴───────────────────────────────────┘
```

## EXECUTE

**프롬프트 (Block 5a에서 진화):**

```
30대 직장인을 위한 AI 트렌드 카드뉴스 5장을 HTML로 만들어줘.
1장: 표지, 2~4장: 본론, 5장: CTA.
1~4장은 제목+핵심 내용 1~2줄+핵심 요약 한 줄,
5장은 제목+핵심 내용+행동 유도 문구.
1080×1080px, 배경색 #6B4EFF, 흰색 텍스트,
헤드라인 15자 이내, Noto Sans KR.
레이아웃:
- 1장(표지): 전체 중앙 정렬, 상하 여백 120px,
             부제는 제목 폰트 크기의 50%
- 2~4장(본문): 좌측 상단에 넘버링(01/02/03),
               제목 상단 고정, 핵심 요약 하단 고정,
               좌우 패딩 80px
- 5장(CTA): 전체 중앙 정렬, CTA 문구 버튼 스타일,
            상하 여백 최대화
↑ 레이아웃 추가
```

**Block 5a 결과와 나란히 비교:**
- 카드 유형별로 배치가 달라졌는지 확인
- "레이아웃을 주면 역할이 생긴다"는 것을 눈으로 확인

**좋은 템플릿의 조건 3가지:**

```
1. 완성형  → 디자인 + 내용이 다 들어있다
             (빈 틀보다 채워진 예시가 훨씬 쓰기 쉽다)

2. 목적별  → 정보전달 / 트렌드 / 홍보
             (목적이 다르면 구조와 디자인이 달라진다)

3. 교체 가능 → 내용만 바꿔도 완성본이 나온다
             (Claude에게 파일 던지면 끝)
```

## EXECUTE

**Step 1 — 3종 템플릿 생성**

```
카드뉴스 HTML 템플릿 3가지를 만들어줘.

공통 스펙:
- 1080×1080px 정사각형, 5장 구성
- 1장: 표지, 2~4장: 본론, 5장: CTA
- 1~4장: 제목+핵심 내용 1~2줄+핵심 요약 한 줄
- 5장: 제목+핵심 내용+행동 유도 문구
- Noto Sans KR 폰트, 헤드라인 15자 이내

template-01-info.html (정보전달형)
- 흰 배경(#FFFFFF), 포인트 컬러 #6B4EFF
- 예시 내용: "ChatGPT 업무 활용법 4가지"
- 각 카드 좌측 상단에 넘버링(01, 02...) 강조

template-02-trend.html (트렌드 리포트형)
- 다크 배경(#1A1A2E), 포인트 컬러 #E94560
- 예시 내용: "2025 AI 트렌드 TOP 4"
- 수치·퍼센트 데이터 강조 스타일

template-03-promo.html (홍보·CTA형)
- 그라디언트 배경(#6B4EFF → #FF6B6B), 흰 텍스트
- 예시 내용: "지금 시작해야 하는 이유"
- 큰 헤드라인, 감성 카피 중심

각 파일을 별도 html 파일로 저장해줘.
```

**Step 2 — 브라우저에서 비교**

```
3개 파일 모두 브라우저에서 열기
체크포인트:
□ 각 템플릿의 목적이 시각적으로 구분되나?
□ 내 채널 주제와 맞는 스타일이 있나?
□ 바로 쓰고 싶은 템플릿이 있나?
```

**Step 3 — 내 주제로 교체 체험**

마음에 드는 템플릿 하나 골라서:

```
template-01-info.html을 참고해서
[내 채널 주제]에 대한 카드뉴스 5장을 HTML로 만들어줘.
디자인 스펙은 그대로 유지하고 내용만 바꿔줘.
```

> 파일을 통째로 던지면 스펙을 다시 설명할 필요가 없다.
> 이게 템플릿의 핵심 가치다.

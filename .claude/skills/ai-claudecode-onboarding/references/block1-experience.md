# Block 1: 7일 후 미리 체험

> **Phase A 시작 시 반드시 아래 형태로 출력한다:**
> ```
> 📖 공식 문서: https://code.claude.com/docs/ko/quickstart
> ```

> 7일 후 모습을 먼저 체험한다. 외우지 않는다. 느끼기만 한다.

## EXPLAIN

먼저 아래 Before/After 테이블을 보여준다:

| 지금 (Before) | 7일 후 (After) |
|---------------|----------------|
| 인스타 캡션 10개 쓰는 데 **2시간** | "이번 주 캡션 10개 써줘" 한 문장 → **10분** |
| 협찬 제안서 쓸 때마다 **1시간 고민** | 브랜드 정보 던지면 초안 즉시, 수정 **10분** |
| 유튜브 썸네일 문구 고르는 데 **30분** | 후보 20개 즉시, 골라서 **3분** |
| "콘텐츠 아이디어 뭐 있어?" → 두루뭉술한 답 | Claude가 내 채널 맥락 파악 후 **딱 맞는 아이디어** |
| 기능 외우려고 문서 뒤지기 | **모르면 Claude에게 바로 물어보기** |

3가지 데모를 안내한다:

---

### 데모 1: Claude가 내 업무를 스스로 파악하며 정리

> 직접 입력해보세요. 실제로 작동합니다.

터미널에서 아래 문장을 그대로 복사해서 입력하세요:

```
나는 인스타그램과 유튜브를 운영하는 1인 크리에이터야.
이번 주에 내가 올린 콘텐츠를 정리하고 다음 주 계획을 세우는 걸 도와줘.
모호한 부분은 AskUserQuestion으로 하나씩 물어봐서 명확하게 만들어줘.
```

Claude가 "어떤 플랫폼을 주로 쓰시나요?", "이번 주 올린 게시물 수는?" 같이 **선택지를 제시하며 질문**하고, 대화를 통해 주간 현황을 정리해줍니다.

**핵심 포인트:** 아직 자동화 스킬이 없어도, Claude와 대화만으로 이 정도가 됩니다. 7일 후에는 이걸 **한 줄 명령**으로 저장해서 매주 쓸 수 있게 됩니다.

---

### 데모 2: 모호한 요청 vs 명확한 요청 차이 체험

먼저 이렇게 **모호하게** 요청해보세요:

```
이번 달 인스타 콘텐츠 전략 만들어줘
```

Claude가 혼자 추측해서 결과를 냅니다.

이번엔 다르게 요청해보세요:

```
이번 달 인스타 콘텐츠 전략 만들어줘. 모호한 부분은 AskUserQuestion으로 질문해서 명확하게 만들어
```

Claude가 "어떤 카테고리 콘텐츠인가요?", "팔로워 수와 타겟은?", "게시 빈도는 주 몇 회?" 같이 **선택지를 제시하며 질문**하는 걸 체험합니다.

**핵심 포인트:** 같은 요청인데 결과물의 깊이가 완전히 달라집니다. 질문 한 줄을 추가하는 것만으로요.

---

### 데모 3: Claude에게 바로 물어보기

```
Claude Code에서 MCP가 뭐야? 크리에이터한테 유용한 예시로 설명해줘
```

**핵심 메시지:** 기능을 다 외울 필요 없습니다. 모르면 Claude에게 물어보면 됩니다. 이 자체가 Claude Code를 쓰는 방식입니다.

---

## EXECUTE

1. **데모 1** 직접 실행: 위 문장을 복사해서 붙여넣기 → Claude가 질문하며 정리해주는 과정 체험
2. **데모 2** 직접 실행: 모호하게 → 명확화 버전 순서로 재요청 → 차이 비교
3. **데모 3** 직접 실행: Claude Code에 대해 뭐든 물어보기

> "위 내용을 직접 실행해보세요. 실행이 끝나면 **'완료'** 또는 **'다음'**이라고 입력해주세요."

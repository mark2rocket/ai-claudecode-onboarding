## ⭐ 필수 3가지 — 오늘 당장 쓸 것

```
┌─ Memory ────── CLAUDE.md(내가 쓴 지시문) + Auto Memory(Claude의 자동 메모)
│
├─ Skill ─────── 반복 업무를 레시피로 저장 → 한 줄로 실행
│
└─ MCP ──────── 인스타·유튜브·노션·슬랙 등 외부 앱 연결 플러그
```

| 기능 | 한 줄 요약 | 크리에이터 활용 |
|------|-----------|--------------|
| **Memory** | 매 대화마다 Claude가 기억하는 나의 지시문 + 자동 메모 | "나는 라이프스타일 크리에이터야"를 매번 말 안 해도 됨 |
| **Skill** | 반복 업무를 저장해두고 한 줄로 실행하는 레시피 | 매주 캡션 작성·리포트 정리를 자동화 |
| **MCP** | 외부 앱을 Claude에 연결하는 플러그 | 노션 콘텐츠 캘린더, 슬랙 팀 메시지를 Claude가 직접 읽고 씀 |

---

## 🔧 심화 4가지 — 나중에 필요할 때

```
├─ Subagent ──── 특정 작업을 독립 공간에서 처리하는 분업 도우미
│   └─ Agent Teams ── 여러 도우미가 팀처럼 동시에 협업
│
├─ Hook ──────── 특정 사건 발생 시 100% 자동 실행되는 규칙
│
└─ Plugin ────── 위의 모든 기능을 묶어서 한 번에 설치·공유
```

| 기능 | 한 줄 요약 |
|------|-----------|
| **Subagent** | 특정 작업만 전담하는 분업 도우미 Claude |
| **Agent Teams** | 여러 Claude가 팀처럼 함께 일하는 협업 시스템 |
| **Hook** | 특정 일이 생기면 자동으로 실행되는 규칙 |
| **Plugin** | 위 기능들을 한 번에 묶어서 설치하는 패키지 |

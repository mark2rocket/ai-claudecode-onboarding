```

```

> GUI [쉬운 말: 앱 화면] — 버튼과 아이콘으로 조작하는 일반적인 프로그램 화면
> CLI [쉬운 말: 명령어 창, 터미널] — 글자를 타이핑해서 컴퓨터에 직접 명령하는 방식

### Quiz 2: 왜 일반 AI 채팅이 아니라 터미널인가?

```json
AskUserQuestion({
  "questions": [{
    "question": "왜 Claude.ai 채팅 같은 일반 AI 도구가 아니라 터미널에서 Claude Code를 써야 할까요?",
    "header": "Quiz 2",
    "options": [
      {"label": "코딩이 모든 지식 노동의 새로운 도구가 될 것이라서", "description": "코딩 = 개발자 전유물이 아니라 글쓰기·기획·마케팅에도 적용"},
      {"label": "Claude.ai 채팅보다 터미널이 더 나아서", "description": "채팅도 좋지만 터미널이 자유도의 끝판왕"},
      {"label": "터미널이 무료라서", "description": "둘 다 유료"},
      {"label": "개발자만 쓸 수 있어서", "description": "비개발자도 씁니다. 지금 여기서."}
    ],
    "multiSelect": false
  }]
})
```

정답: **1번**. 해설:

```
Claude.ai 채팅 → 정해진 대화창 안에서의 생산성
터미널 Claude Code → 컨텐츠 제작·업무 자동화라는 새로운 차원의 일

토큰을 많이 쓸 수 있다 = 엄청나게 많은 작업 인력을 가진 사람이 된다
```

> 토큰 [쉬운 말: AI가 처리하는 글자 단위] — Claude가 읽고 쓰는 텍스트의 기본 단위.
> 토큰을 많이 쓸 수 있다는 건, 그만큼 Claude에게 더 많은 일을 시킬 수 있다는 의미입니다.

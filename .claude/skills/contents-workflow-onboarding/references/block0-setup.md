## EXPLAIN

### Claude Code란?

Claude Code는 **터미널** [쉬운 말: 명령어 창 — 마우스 클릭 대신 글자를 입력해서 컴퓨터와 대화하는 창]에서 Claude AI를 사용하는 도구입니다.

채팅창에서 대화하는 것과 달리, 터미널에서는 Claude가 **실제로 파일을 만들고, 앱을 실행하고, 외부 서비스와 연결**하는 일을 할 수 있습니다.

---

### 에디터 선택 (선택 사항)

**에디터** [쉬운 말: 코드/글 편집 프로그램]는 없어도 되지만, 있으면 Claude Code와 함께 쓰기 편합니다.

| 에디터 | 특징 | 추천 대상 |
|--------|------|-----------|
| **Antigravity** | 무료, AI 내장, 가장 쉬움 | 완전 처음인 분 |
| **Cursor** | AI 내장, Claude Code와 궁합 최고 | 조금 써본 분 |
| **VSCode** | 범용적, 확장 기능 많음 | 이미 쓰고 있는 분 |

---

### 초기 설정 3단계

**1단계:** 터미널에서 `claude` 입력 후 실행
```
claude
```
> 처음 실행하면 Anthropic 계정 로그인 화면이 나옵니다.

**2단계:** Claude Pro/Max 구독을 연결합니다.

**3단계:** Output Style을 "Explanatory"로 설정합니다.
```
/output-style
```
> Explanatory [쉬운 말: 설명형] 모드를 선택하면 Claude가 무슨 일을 하는지 친절하게 설명해줍니다.

---

### Mac 사용자 꼭 알기

Mac에서는 `.`으로 시작하는 폴더(예: `.claude`, `.git`)가 **기본적으로 숨겨져서 보이지 않습니다.**

숨겨진 폴더가 안 보인다면 Claude Code에 이렇게 요청하세요:
```
Mac에서 숨겨진 폴더(.로 시작하는 것들)가 보이게 해줘
```
